# Job-Finder 
- Esse site foi desenvolvido durante a finalização de um curso inicial de JavaScript. Dentro dele foi possível aprender conceitos relacionados a Front e Backend. 

#### Qual o intuito do site? 

- O site tem, como principal objetivo, o cadastro e a busca de emprego para profissionais da área de TI.

### :computer: Tecnologias:

- Node.js
- JavaScript 
- HTML
- CSS

### Como acessar?

```bash
  # Clone esse repositório
  $ git clone https://github.com/IzabellaRibeiro/Job-Finder.git
  
  #Instale as dependências
  $ npm install
  
  #Inicie
  $ npm run dev
